DEPLOY
1. Upload index.html (with the assets/ folder if you want backup logo files) to your web host.
2. The HTML is self-contained — every logo is embedded as base64. Works standalone.
3. Compatible with Vercel, Netlify, Cloudflare Pages, GoDaddy, AWS, etc.

BEFORE GOING LIVE — QUICK CUSTOMIZATIONS
1. Replace placeholder phone (800) 555-1234 with your real number
   (search for "1234" in index.html — appears in 5 places)

2. Wire up form submission
   The full-offer-form currently logs to console. Replace the body of
   the submit handler near the end of <script> with a real fetch() POST
   to your CRM endpoint (HighLevel, REIPro, Podio, etc.) or a webhook
   service like Zapier/Make.

3. Replace placeholder testimonial names/initials with real ones once
   you have them. Located in the "TESTIMONIALS" section.

4. Add Google Analytics / Meta Pixel / call tracking inside <head>.

5. Update the canonical URL and og:url meta tags from silverlynxhomes.com
   if your domain differs.

6. Replace the (800) 555-1234 with your dialer/tracked phone number once
   set up.
